# AI Assistant

An offline, privacy-focused AI assistant with mood simulation.

## Features
- Web search and fact-checking
- Mood system with safeword ("epsilon")
- Long-term memory

## Installation
```
bash
git clone https://github.com/Kofse/MynahSim.git
cd ai-assistant
chmod +x install.sh
./install.sh
```
